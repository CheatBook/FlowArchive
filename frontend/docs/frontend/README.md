# Flow Archive フロントエンド・ドキュメント

このフォルダには、フロントエンド（React/TypeScript）の仕組みを理解するための詳細な解説書が格納されています。

## 📂 フォルダ構成

- **`main.md`**: アプリケーションの起動と初期設定の解説。
- **`App.md`**: ルーティング（画面遷移）と全体レイアウトの解説。
- **`components/`**: 共通部品の解説。
  - `Sidebar.md`: サイドバーメニュー。
  - `MarkdownRenderer.md`: Markdown表示とコードの色付け。
- **`pages/`**: 各画面の解説。
  - `KnowledgeList.md`: 一覧表示とページネーション。
  - `KnowledgeDetail.md`: 詳細表示。
  - `KnowledgeForm.md`: 新規作成・編集フォーム。

## 💡 学習の進め方

1. まずは `main.md` と `App.md` を読み、アプリの全体像を把握しましょう。
2. 次に `pages/KnowledgeList.md` を読み、データの取得と表示の流れを理解します。
3. 最後に `components/` 内の部品がどのように再利用されているかを確認してください。

各ドキュメントは、実際のコードを一行ずつ解説する形式になっています。エディタでコードを開きながら読み進めることをお勧めします。
